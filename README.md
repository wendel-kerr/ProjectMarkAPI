# Phase 6 — CRUD de Resources + árvore com resources

Endpoints:
- POST   /resources
- GET    /resources/:id
- GET    /resources?topicId=<id>
- PATCH  /resources/:id
- DELETE /resources/:id

- GET    /topics/:id/tree?includeResources=true|false&version=latest|<n>

Quickstart:
```
npm install
npm run dev
# ou
npm run build && npm start
```
